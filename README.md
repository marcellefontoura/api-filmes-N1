# API de Filmes

Este projeto foi desenvolvido com o objetivo de colocar em prática os conceitos básicos de back-end por meio da criação de uma API de filmes.

A aplicação simula um pequeno sistema de cadastro e listagem, permitindo entender melhor como funciona a comunicação entre cliente e servidor em uma aplicação back-end.

## Objetivo

Consolidar os principais conceitos estudados até aqui, como:

- criação de rotas
- uso dos métodos HTTP `GET` e `POST`
- utilização do middleware `express.json()`
- uso de parâmetros de rota
- testes de endpoints com Postman

## Funcionalidades

A API foi construída para permitir:

- listar filmes cadastrados
- cadastrar um novo filme
- buscar um filme específico pelo ID

## Conceitos aplicados

Durante o desenvolvimento deste projeto, foram trabalhados os seguintes pontos:

- **Rotas** para organizar os endpoints da aplicação
- **GET** para buscar ou listar filmes
- **POST** para cadastrar novos filmes
- **Middleware `express.json()`** para interpretar dados em JSON
- **Parâmetros de rota** para buscar filmes específicos, como em `/filmes/:id`
- **Postman** para testar o funcionamento das rotas
